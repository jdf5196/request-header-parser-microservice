Request Header Parser Microservice API

What does this do?

This is a service that returns the users IP address, language and operating system.

User Stories

  1. I can get the IP address, language and operating system for my browser.

Examples:

{
  ipaddress: "xxx.xx.xx.x",
  language: "en-US",
  software: "Macintosh; Intel Mac OS X 10_10_5"
}
